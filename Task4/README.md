Task 4 - Rock Paper Scissors Game

## Overview ##

This project is a simple command-line based Rock–Paper–Scissors game built using Python.
The user plays against the computer, and the program decides the winner based on the classic rules of the game.

## Features ##

1.User chooses Rock, Paper, or Scissors

2.Computer randomly selects its move

3.Game decides win, lose, or tie

4.Score tracking for both user and computer

5.Option to play multiple rounds

6.Simple text-based interface

## How It Works ##

The program uses basic if–else logic to compare the user’s choice with the computer’s choice.
The rules followed are:

~ Rock beats Scissors

~ Scissors beat Paper

~ Paper beats Rock

Scores update after every round, and the user can choose to continue or exit the game.

## Requirements ##

~ Python installed on your system (Python 3 recommended)

## How to Run ##

1.Open the project folder

2.Open a terminal or command prompt

3.Run the command:python rps.py


4.Choose Rock, Paper, or Scissors when prompted

5.Continue playing or exit when asked

## Notes ##

A. The game resets the score when you restart the program.
B. This project is designed for beginners to practice Python input handling, randomness, and basic conditional logic.
