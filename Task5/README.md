Task 5 - Contact Book

## Overview ##

This project is a simple command-line based Contacts Manager built using Python.
It allows users to store and manage basic contact information quickly and easily.

## Features ##

1.Add new contacts

2.View all saved contacts

3.Search contacts by name or phone number

4.Update existing contact information

5.Delete a contact

6.Simple menu-driven user interface

## How It Works ##

The program stores contacts in a list while it is running.
Each contact contains:

~ Name

~ Phone Number

~ Email

~ Address

Users interact with the application through a numeric menu to manage contact details.

## Requirements ##

~ Python installed on your system (Python 3 recommended)
